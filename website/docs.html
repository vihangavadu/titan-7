Quick Start Guide

Get Titan OS V9.1 deployed and operational. This guide covers deployment from the bootable ISO or VPS installation.

1

Obtain Titan OS License

Contact Lucid Empire to obtain your Titan OS license and receive the V9.1 ISO image (2.7GB).

2

Deploy the ISO

Terminal 
# Boot from ISO (bare metal or VM)
# OR deploy to VPS via SSH
scp titan-os-9.1.iso root@your-vps-ip:/tmp/
ssh root@your-vps-ip
bash /opt/titan/scripts/install_titan.sh
3

First Boot Setup

Terminal 
# Titan OS auto-configures on first boot:
# - Redis, Ollama, Xray, ntfy services start
# - 6 AI models load into Ollama
# - XFCE desktop + XRDP remote access enabled
# - All 115 core modules verified

# Verify installation:
python3 /opt/titan/verify_all.py
# Expected: OK:113 FAIL:0
4

Launch the GUI

Terminal 
# Connect via XRDP (port 3389) or local desktop
# Launch the Titan OS 3x3 app launcher:
python3 /opt/titan/apps/titan_launcher.py

# Or launch individual apps:
python3 /opt/titan/apps/titan_operations.py
python3 /opt/titan/apps/titan_intelligence.py
python3 /opt/titan/apps/app_browser_launch.py

VPS Setup

Deploy Titan OS to any Debian 12 VPS. Tested on Hostinger KVM 8 (8 CPU, 32GB RAM, 400GB disk).

Terminal
# 1. SSH into your VPS
ssh root@your-vps-ip

# 2. Upload and extract Titan OS
scp titan_deploy.zip root@your-vps-ip:/opt/
cd /opt && unzip titan_deploy.zip

# 3. Run the installer
bash /opt/titan/scripts/install_titan.sh

# 4. Install external dependencies
# Ollama, Camoufox, Mullvad VPN, Xray, Redis, ntfy
bash /opt/titan/scripts/install_self_hosted_stack.sh

# 5. Load AI models
bash /opt/titan/modelfiles/deploy_models.sh

# 6. Verify
python3 /opt/titan/verify_all.py
Note: Titan OS requires root access for kernel module installation and network stack modifications. VPS must run Debian 12 with kernel 6.1+.

8-Ring Defense Architecture

Titan OS uses a concentric ring defense model. Each ring adds a layer of protection, from hardware to operator.

Ring 1 — Network Shield    → TLS parroting, JA4 rotation, VPN, proxy
Ring 2 — Environment       → Canvas, WebGL, font, audio, timezone synthesis
Ring 3 — Identity/Profile  → Genesis profile engine, behavioral biometrics

API Reference

Titan OS exposes 59 internal API endpoints through titan_api.py. The API serves as the bridge between all modules and the GUI layer.

Python
from core.titan_api import TitanAPI

api = TitanAPI()

# Generate a new profile via Genesis engine
profile = api.genesis_create_profile(
    country="US",
    age_range="25-35",
    gender="male"
)

# Run AI analysis
result = api.ai_analyze(
    task="risk_assessment",
    data=profile
)

# Check system health
status = api.health_check()
print(f"Modules: {status.modules_ok}/115")

Frequently Asked Questions

What VPS provider is recommended?

Titan OS is tested on Hostinger KVM 8 (8 CPU, 32GB RAM, 400GB disk) with Debian 12. Any KVM VPS with 4+ cores and 4GB+ RAM running Debian 12 will work.

Are the AI models trained on custom data?

Yes. The 3 custom Ollama models (Analyst, Strategist, Fast) are fine-tuned on 2,200+ training examples specific to Titan OS operations. The ONNX engine uses INT4 quantization for fast inference.

Can I customize the GUI applications?

Yes. All 9 apps are written in PyQt6 and are fully extensible. The Enterprise license includes source code access for custom modifications.

How do I get support?

Contact Lucid Empire directly. All licenses include email support. Professional and Enterprise tiers include priority support from Lucid.
